To run all algorithms please install R. Found here: http://mirrors.nics.utk.edu/cran/. 
Once R is installed please open R and in the "Packages" menu select "Install Packages..."
Select a mirror and install the following packages: adabag, caret, class, e1071, neuralnet, and tree
After installing these algorithms open the desired algorithm by selecting "Open Script..." in the "File" menu
On the first line of the script change the directory to the "Datasets" directory on your local machine
Click the "run line or selection" button, the third button, to execute the commands in the script. Execute these one line at a time.
